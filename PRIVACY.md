# Privacy Policy

This Privacy Policy applies to the New Tab web extension (the "Application") for web browsers, developed by Shane (the "Developer") as an open-source project licensed under the [MIT License](LICENSE). By using the Application, you agree to the terms described in this Privacy Policy.

### Data Collection

The Developer does not collect, store, or retain any personal data. When you provide a GitHub API key the Application sends requests directly from your browser to the GitHub API. The Developer never receives, processes, or has access to any data.

### Data Storage

The Application does not store any data on external servers. API results are stored on your browser's local storage and persists until you clear the data. The Application's settings, **including your API key**, are stored in your browser's web extension sync storage, which may sync across all browsers and devices linked to your browser's account. This synced data may be written to your device's file system in plain text.

### Third-Party Services

The Application sends requests to the GitHub API if an API key is provided. The Developer has no control over how any third-party service receives, processes, stores, or uses that data. Please review the applicable policies below:
