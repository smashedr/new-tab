[Chrome Web Store](https://chromewebstore.google.com/detail/goholgjajaneidafddjpiaphbdaalbhm).

- Download a [Chrome Release](https://github.com/cssnr/new-tab/releases/latest/download/new-tab-chrome.crx) from GitHub
- Download a [Firefox Release](https://github.com/cssnr/new-tab/releases/latest/download/new-tab-firefox.xpi) from GitHub

## Features

- GitHub Repository List from History
- Add/Edit Custom Search Queries
- Customize the Look and Feel
- Select a Bookmarks Folder
- Optional Top Sites

> [!TIP]  
> **Don't see your feature?**
> Submit a [Feature Request](https://github.com/cssnr/new-tab/issues/new?template=1-feature.yaml).

Firefox: toggling Debug logs, Chrome: toggling Verbose from levels dropdown.
